This is a work-in-progress decompilation of Halo: Combat Evolved build 2342 (`cachebeta.exe`, sha256 `4cc87b45f721270392a96f1674ed2b5cd4a7bb4355faeab4531d1cf1884d9520`).

You must source the August 2001 Xbox SDK yourself and extract the `XDK/xbox` folder from the installer into the repository root such that `xbox/{bin,include}` are valid paths.
